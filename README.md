# Fintech-Data-Analysis
Analysis the financial data and understand the business problems and solve this business questions using SQL and PowerBI

“End-to-End Fintech Risk, Customer & Banking Analytics System”
________________________________________
🧾📌 Project Overview &Explaination
👉“In this project, I worked on a fintech dataset consisting of customers, transactions, accounts, and loans to analyze customer behavior, detect fraud, and assess financial risk using SQL and Power BI.”
👉“The goal was to build a complete customer-level intelligence system combining transaction activity, banking data, and loan information.”

Dataset Overview (4 Tables)
1. customer table
2. Transaction Table
3. Account Table
4. Loan table
   
🎯 Business Objectives
👉“The project focused on solving real fintech problems:”
1.	Identify high-risk customers
2.	Detect and analyzefraud patterns
3.	Understand customer financial behavior
4.	Evaluate loan risk and defaults
5.	Build a customer 360 view
